## 🧬 为什么选择鬼面？

### 🧠 三层记忆架构 — 不只是"自动总结"

大多数总结脚本只做一件事：让 AI 把聊天内容塞进一个表格。**鬼面完全不同。**

```
                         ┌─────────────────────────┐
                         │   📜 Layer 3: 大总结     │  ← 全局叙事报告（2000+ 字）
                         │   结构化档案 · 情感递进   │    自动编号 · 旧条目归档
                         └────────────┬────────────┘
                                      │
              ┌───────────────────────┼───────────────────────┐
              │                                               │
   ┌──────────▼──────────┐                       ┌────────────▼───────────┐
   │ 🧩 Layer 1: 记忆碎片 │                       │ 📅 Layer 2: 故事时间线  │
   │ 细粒度情报提取        │                       │ 全局剧情大纲追踪        │
   │ 分类标签 · 关键词触发  │                       │ 生成 → 合并 → 压缩      │
   └──────────────────────┘                       └──────────────────────┘
```

| 能力 | 鬼面 | 普通总结脚本 |
|------|------|-------------|
| 记忆层级 | **3层**（碎片 → 时间线 → 大总结） | 1层（直接填表） |
| 去重机制 | **5级管线**（NLP去重 + 标题模糊匹配 + 主动合并 + 跨语言语义 + 编辑距离） | 纯靠 prompt 让 AI 自己判断 |
| 时间线追踪 | ✅ 独立系统（自动生成 / 合并 / 压缩） | ❌ |
| 记忆触发 | ✅ 每条碎片带 3-8 个关键词，按需激活 | ❌ 全量注入 |
| 上下文管理 | ✅ 自动 Token 估算 + 超限压缩 | 部分支持 |

## 💎 核心特性

### 🧩 记忆碎片系统
不再生成一大块总结文本。鬼面将对话中的关键信息提取为**独立的记忆碎片**——每条碎片带有分类标签（如 `[喜好-热可可]`、`[事件-坦白恐惧]`）和 **3-8 个中英双语触发关键词**，存入世界书后按需激活，精准模拟人脑的联想记忆。

**智能去重 + 合并**：鬼面在代码层面实现了 5 级去重合并管线：
1. 🔄 鬼面主动合并（`===UPDATE===` 格式，覆盖旧条目 + 合并关键词）
2. 🏷️ 标题模糊匹配（相似度 >60% 自动合并，防止近似条目堆积）
3. 🔤 标点清除 + 同义词归一化（中英双语：`喜欢/喜爱/钟爱` → `like`）
4. 🌍 中英文跨语言语义相似性检测
5. ✏️ 编辑距离算法（Levenshtein Distance）

### 📅 故事时间线
全新的独立时间线引擎（`timeline.js`），为 LLM 提供全局剧情脉络——**再也不会"失忆"**。

- **智能时间标注**：自动从对话推断日期（`[第1天-清晨]`、`[2024.6.15 下午]`）
- **多片段合并**：高楼层总结时逐段生成时间线片段，最终用 LLM 合并为连贯大纲
- **自动压缩**：超过 2000 tokens 时，自动将早期事件压缩为章节概括，永不占满上下文
- **世界书常驻**：作为常驻条目注入世界书，LLM 每轮对话都能看到

### 📜 大总结系统
不是普通的"概括一下聊天内容"。鬼面的大总结生成**结构化深度报告**：

- 📅 **时间锚点** — 精确标注故事发生的时间节点
- 🔥 **情节发展** — 至少 2000 字的深度剧情梳理
- ❤️ **情感递进** — 角色间关系变化的关键词追踪
- 🧠 **关键档案同步** — 永久性事实更新、物品/地点追踪
- 🧩 **世界线索** — 未回收的伏笔、隐性机制记录

自动编号系统（P-001, P-002...），旧大总结自动归档，世界书始终整洁。

### 🏗️ 高楼层总结（万楼补课）
上万楼从没总结过？**一键补课**：按分段大小（10-100 楼/段）自动切分，逐段执行：

```
对每一段 → 时间线片段生成 → 记忆碎片提取 → 下一段...
         → 全部完成后合并时间线 → 写入世界书 → 可选隐藏
```

## 📱 虚拟手机 — App+社区生态圈，完整的货币系统

鬼面内置了一部 **手机**，拥有独立的工作系统、悬浮入口、iOS 通知推送，和 8 款完整的 App（未来会更新更多）。手机可以配置使用独立 API 或主 LLM，所有 App 共享同一个上下文。

> 📐 手机模块：**50+ 个文件** · **~900KB 代码**

### 📸 朋友圈
独立的社交模拟系统，搭载服务器后端（SQLite 数据库）：

- **真人社交网络**：注册后获得独立 ID，可以添加群友为好友
- **你对象互动**：你和你的你对象发的动态，好友和好友的角色都可以互动/回复/点赞
- **自动发帖**：你对象根据聊天剧情自动生成朋友圈动态，好友互动实时同步
- **多媒体卡片**：原生支持 `<图片>`、`<视频>`、`<音乐>`、`<新闻>` 标签自动渲染
- **世界书联动**：所有朋友圈事件实时静默同步至世界书，让 LLM 拥有完整的"社交上下文"
- **iOS 推送通知**：在你使用手机时，新动态和评论会以通知横幅弹出

### 💬 聊天
**iMessage 风格**的独立聊天 App：

- **私密聊天空间**：异地时聊天，和主剧情丝滑衔接。本质上是恶灵短信版超进化。
- **多条待发**：🥝 猕猴桃按钮支持攒多条消息一次发送
- **特殊消息**：语音描述 / 图片描述 / 分享卡片 / 道具使用
- **表情回应**：长按或双击气泡弹出 emoji 反应选择器
- **你对象内心想法**：点击气泡可展开/折叠内心独白
- **道具背包**：在聊天中直接使用商城购买的道具
- **🏠 我已回家**：一键总结今日聊天，自动发送至主对话，结束异地
- **批量删除**：iMessage 风格的多选删除模式
- **重新生成**：对最后一条回复重新生成

### 📔 日记本
**情侣交换日记**——和你对象一起记录每天的故事：

- **双人视角**：你写下心情和内容后，你对象用 ta 的视角回应你的日记
- **每日续写**：同一天多次书写自动追加到当天日记
- **情绪表达**：12 种心情 emoji 可选
- **标签系统**：自定义标签归类日记
- **🔍 搜索**：按关键词全文搜索所有日记
- **📅 日历视图**：日历浏览器，有日记的日期高亮显示
- **🎨 深度自定义**：4 款预设主题（暖阳/樱花/薰衣草/墨绿）+ 完全自定义颜色、字体（含 Google Fonts）、字号、行高
- **🖨️ 打印**：精选日记可打印珍藏
- **自动模式**：日记内容自动注入世界书
- **多层持久化**：chat_metadata + extension_settings + localStorage 三级存储
- **两种模式**：在设置页面，可自行选择自己喜欢的。

### 🏪 商城 (Shop)
**Amazon 风格**的虚拟商城：

- **暗金细胞**：社区虚拟货币系统，通过社区活动赚取
- **分类浏览**：商品按分类排列，商品卡片展示详情
- **物品库**：购买的道具进入物品库，可在聊天中使用
- **道具效果**：聊天 Prompt 注入 / 日记 Prompt 注入 / 性格覆盖 / 特殊消息等多种效果类型
- **⭐ 评价系统**：用户可写评价，更可让你对象以被使用者视角写评价（LLM 生成）
- **🎁 送礼系统**：你对象有概率在聊天中自动送礼物，真实可用的宠物/dbd游戏道具，鬼面bot会私信你提醒
- **🎭 恶作剧系统**：使用恶作剧道具整蛊你对象
- **💀 抢劫系统**：你对象尝试抢劫其她用户的暗金细胞，可能被反杀！

### 🔮 占卜
来自黎明杀机的两位灵媒——**Sable Ward** 和 **Mikaela Reid**，被恶灵从迷雾中派来为你进行塔罗占卜：

- **塔罗牌图片**：完整的塔罗牌图片资源

- **刷新即逝**：对话不持久化，刷新页面自动消失——占卜的神秘之处

## 🚀 快速上手

1. 将本扩展文件导入插件面板
2. 在酒馆中启用扩展
3. 打开角色聊天，鬼面面板会自动加载
4. **推荐**：在"自定义 API 配置"中单独设置总结用模型，避免占用主对话 API

### 触发设置
- Token 和楼层数满足任一条件即触发自动总结
- 也可随时通过面板按钮手动触发

### 手机使用
- 点击聊天界面上的悬浮手机图标，或控制台的「打开手机」按钮
- 手机可以配置使用独立 API（与鬼面总结共用）或主 LLM

### 查看结果
- 总结完成后打开世界书查看生成的记忆碎片 / 时间线 / 大总结条目
