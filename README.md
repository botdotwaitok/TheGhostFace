## 这是什么

给 SillyTavern 的全栈式扩展，三件套：

- **三层记忆架构** — 记忆碎片 / 故事时间线 / 大总结，自动写入世界书
- **虚拟手机** — 内嵌一部带十几款 App 的手机，可独立 LLM
- **朋友圈服务器** — Express + SQLite，跨用户社交模拟
